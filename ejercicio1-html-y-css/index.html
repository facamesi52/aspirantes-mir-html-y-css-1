Receta: Ensalada

Ingredientes

Los ingredientes necesarios para preparar la ensalada son los siguientes:

  • Pechuga
  • Lechuga
  • Tomate
  • Cebolla
  • Maiz

Preparacion

Para preparar una ensalada se siguen los siguientes pasos:

  1. Se lavan los ingredientes cuidadosamente
  2. Se sofrie la pechuga y se corta en cubos
  3. Se cortan al gusto los vegetales
  4. Se incorporan todos los ingredientes
  5. Servir lo mas fresco posible
